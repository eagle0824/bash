这个脚本是搭建ubuntu16.04下开发环境，安装一些必要的工具以及修改部分环境配置


目录结构如下

├── android             //android usb设备相关配置
├── git_config.sh       //git 相关配置，需要修改 user.name和user.email
├── imotor              //imotor相关的工具，文档仓库，此执行之前需要配置ssh-key，确保有权限获取code
├── java                //jdk切换工具的配置
├── pc_init.sh          //程序的入口，更新源,安装基本的工具
├── Readme.txt          //帮助文档
├── repo                //repo相关配置
├── vim                 //vim相关的配置 
└── workspace.sh        //开发环境的搭建
